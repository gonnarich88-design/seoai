    ```typescript
    export interface BrandMatch {
      brandId: string;
      mentioned: boolean;
      position: number | null;       // ordinal rank among all detected brands (1-based)
      contextSnippet: string | null; // 50-100 chars around mention
    }

    export interface BrandInput {
      id: string;
      name: string;
      aliases: string[];
    }

    export function detectBrands(responseText: string, brands: BrandInput[]): BrandMatch[]
    ```

    Implementation:
    1. For each brand, search `responseText.toLowerCase()` for `brand.name.toLowerCase()` and each alias.
    2. Track the earliest character index where any search term matches.
    3. If no match found: `{ brandId, mentioned: false, position: null, contextSnippet: null }`.
    4. Collect all brands with matches, sort by their earliest character index.
    5. Assign ordinal position: the brand appearing first gets position=1, second gets position=2, etc.
    6. Brands not mentioned get position=null.
    7. Extract context snippet using `extractSnippet(responseText, matchIndex, matchLength)`:
       - Take 40 chars before and after the match.
       - Prepend `...` if start > 0, append `...` if end < text.length.

    ```typescript
    import { generateText } from 'ai';
    import { openai } from '@/lib/ai/providers';

    export type Sentiment = 'positive' | 'neutral' | 'negative';

    export async function analyzeSentiment(contextSnippet: string): Promise<Sentiment> {
      const result = await generateText({
        model: openai('gpt-4o-mini'),
        prompt: `Classify the sentiment of this brand mention as exactly one of: positive, neutral, negative.\n\nContext: "${contextSnippet}"\n\nRespond with only one word: positive, neutral, or negative.`,
      });

      const sentiment = result.text.trim().toLowerCase();
      if (sentiment === 'positive' || sentiment === 'neutral' || sentiment === 'negative') {
        return sentiment;
      }
      return 'neutral'; // fallback for unexpected responses
    }
    ```

    Export `aggregateSnapshot(batchId: string): Promise<void>`. Implementation:
    1. Query all `queryRuns` WHERE `batchId = batchId`.
    2. If no runs, return early.
    3. Extract `keywordId`, `providerId` from the first run.
    4. Get today's date as `YYYY-MM-DD` string.
    5. Get all `brandMentions` for the run IDs using `sql\`${brandMentions.queryRunId} IN ${runIds}\``.
    6. Group mentions by `brandId`.
    7. For each brand group:
       - `mentionRate = mentionedCount / runs.length`
       - `avgPosition = average of position values where mentioned=true and position is not null`
       - Upsert to `dailySnapshots` using `db.insert(...).onConflictDoUpdate({ target: [dailySnapshots.keywordId, dailySnapshots.brandId, dailySnapshots.providerId, dailySnapshots.date], set: { mentionRate, avgPosition, runCount } })`.
    8. Use `parseFloat(mentionRate.toFixed(4))` and `parseFloat(avgPosition?.toFixed(2))` for decimal precision.

    Export `handleQueryJob(job: Job): Promise<void>`. Implementation:
    1. Extract `{ keywordId, providerId, batchId, runNumber, prompt, promptVersion, runType }` from `job.data`.
    2. Call `executeQuery(providerId, prompt)` from `@/lib/pipeline/query-executor`.
    3. Insert into `queryRuns` table:
       ```typescript
       const [queryRun] = await db.insert(queryRuns).values({
         keywordId,
         providerId,
         model: MODELS[providerId],
         prompt,
         promptVersion,
         rawResponse: result.text,
         citations: result.citations,
         tokensUsed: result.totalTokens,
         costUsd: result.costUsd.toFixed(6),
         latencyMs: result.latencyMs,
         runNumber,
         batchId,
         runType,
       }).returning();
       ```
    4. Call `addAnalysisJob(queryRun.id)` to trigger brand detection.
    5. Check if batch is complete: `SELECT COUNT(*) FROM query_runs WHERE batch_id = batchId`. If count === 3, call `addSnapshotJob(batchId)`.

    **Step 2: Create src/worker/handlers/analysis-handler.ts**

    Export `handleAnalysisJob(job: Job): Promise<void>`. Implementation:
    1. Extract `{ queryRunId }` from `job.data`.
    2. Load the queryRun from DB: `db.select().from(queryRuns).where(eq(queryRuns.id, queryRunId))`.
    3. Load ALL brands from DB: `db.select().from(brands)`.
    4. Call `detectBrands(queryRun.rawResponse, brands.map(b => ({ id: b.id, name: b.name, aliases: b.aliases ?? [] })))`.
    5. For each `BrandMatch` result, insert into `brandMentions`:
       ```typescript
       const [mention] = await db.insert(brandMentions).values({
         queryRunId,
         brandId: match.brandId,
         mentioned: match.mentioned,
         position: match.position,
         contextSnippet: match.contextSnippet,
       }).returning();
       ```
    6. If `match.mentioned === true`, call `addSentimentJob(mention.id)`.

    **Step 3: Create src/worker/handlers/sentiment-handler.ts**

    Export `handleSentimentJob(job: Job): Promise<void>`. Implementation:
    1. Extract `{ brandMentionId }` from `job.data`.
    2. Load the brandMention: `db.select().from(brandMentions).where(eq(brandMentions.id, brandMentionId))`.
    3. If `mention.contextSnippet` is null or `mention.mentioned` is false, return early.
    4. Call `analyzeSentiment(mention.contextSnippet)`.
    5. Update the brandMention: `db.update(brandMentions).set({ sentiment }).where(eq(brandMentions.id, brandMentionId))`.

    **Step 4: Create src/worker/handlers/snapshot-handler.ts**

    1. Extract `{ batchId }` from `job.data`.
